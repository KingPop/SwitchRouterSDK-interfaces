This function creates/destroy/edit a policer in the system.

Parameters:
[in]handle- SX-API handle
[in]cmd- SX_ACCESS_CMD_CREATE: create a new policer and returns a policer ID in policer_id_p SX_ACCESS_CMD_DESTROY: destroy an existing policer (according to policer_id_p) SX_ACCESS_CMD_EDIT: edit configuration parameters of an existing policer
[in]policer_attr_p- policer attributes. Ignored when cmd is DESTROY.
[in,out]policer_id_p- policer ID
Returns:
SX_STATUS_SUCCESS if operation completes successfully
SX_STATUS_PARAM_NULL if pid_p or policer_attrib_p pointers are NULL
SX_STATUS_INVALID_HANDLE if vpi_handle is invalid
SX_STATUS_CMD_UNSUPPORTED if cmd is SX_ACCESS_CMD_DESTROY or any other unsupported command
SX_STATUS_PARAM_ERROR if any of the input parameters is invalid
SX_STATUS_COMM_ERROR if an error occured on send or receive message to driver
SX_STATUS_ERROR general error
