This function creates/destroy/edit a policer in the system. Supported devices: SwitchX, SwitchX2, Spectrum.

Parameters:
[in]handle- SX-API handle
[in]cmd- SX_ACCESS_CMD_CREATE: create a new policer and returns a policer ID in policer_id_p SX_ACCESS_CMD_DESTROY: destroy an existing policer (according to policer_id_p) SX_ACCESS_CMD_EDIT: edit configuration parameters of an existing policer. only CIR, EIR, EBS, CBS, yellow action and red action can be edited
[in]policer_attr_p- policer attributes. Ignored when cmd is DESTROY.
[in,out]policer_id_p- policer ID
Returns:
SX_STATUS_SUCCESS if operation completes successfully
SX_STATUS_CMD_UNSUPPORTED if cmd is SX_ACCESS_CMD_DESTROY or any other unsupported command
SX_STATUS_PARAM_ERROR if any of the input parameters is invalid
SX_STATUS_ERROR general error
SX_STATUS_NO_RESOURCES if there are no more resources
SX_STATUS_RESOURCE_IN_USE if trying to destroy a binded policer
SX_STATUS_WRONG_POLICER_TYPE if wrong policer_type was supplied
